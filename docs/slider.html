Slider

The slider component to show progress. you should install and link react-native-reanimated and react-native-gesture-handler to be able to use it. All animated values must be imported from react-native-reanimated.

Usage

import Slider from 'react-native-reanimated-slider';
...

renderBallon=()=>(
 <View>
   <TextInput ref={this.text} />
 </View>
)

setBallonText=(text)=>{
  this.text.setNativeProps({text})
}

render(){
  return (
    <Slider
      style={{ flex: 1 }}
      minimumTrackTintColor="#fff"
      thumbTintColor="#fff"
      ballon={value => this.convertSecondToTime(value)}
      progress={this.currentTime}
      min={new Reanimated.Value(0)}
      cache={this.playableDuration}
      max={this.seekableDuration}
      onSlidingStart={this.slidingStart}
      onSlidingComplete={this.slidingComplete}

      // only if you want to render custom ballon for sliding
      renderBallon={this.renderBallon}
      setBallonText={this.setBallonText}
    />
  )
}

Props

minimumTrackTintColor
Type: string
Default value: "#f3f"
maximumTrackTintColor
Type: string
Default value: "transparent"
cacheTrackTintColor
Type: string
Default value: "#777"
style
Type: any

style for the container view

borderColor
Type: string
Default value: "#fff"

color of the border of the slider

ballon (required)
Type: number => string

a function that gets the current value of the slider as you slide it, and returns a string to be used in the ballon

progress (required)
Type: Animated.Value

an AnimatedValue from react-native-reanimated library which is the current value of the slider.

cache
Type: Animated.Value

an AnimatedValue from react-native-reanimated library which is the curren value of the cache. the cache is optional and will be rendered behind the main progress indicator.

min (required)
Type: Animated.Value

an AnimatedValue from react-native-reanimated library which is the minimum value of the slider.

max (required)
Type: Animated.Value

an AnimatedValue from react-native-reanimated library which is the maximum value of the slider.

onSlidingStart (required)
Type: () => void

callback called when the users starts sliding

onSlidingComplete (required)
Type: number => void

callback called when the users stops sliding. the new value will be passed as argument

renderBallon
Type: () => React.ReactNode

render custom Ballon to show when sliding.

setBallonText
Type: string => void

this function will be called while sliding, and should set the text inside your custom ballon.
